## Unsplash Clone
- simple app to practice with api, react-router



# Installation

1. Clone this repo

```sh
git clone https://github.com/1991Pawel/unsplash-clone.git
```

2. Install NPM packages

```sh
npm install 
```
3. Start a development server
npm 

```sh
npm run start
```
